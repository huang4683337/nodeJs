# 目录结构
## 1、目录结构

```shell
|-- dateFile                        存放和处理数据的文件夹
    |-- js 下的 dataHandle.js        增、删、改、查 功能的封装实现(回调函数)
    |-- json 下的 db.json            用来存储数据                         
|-- views                           用来存放视图文件
|-- index.js					    入口文件
|-- touter.js       				路由配置
```
